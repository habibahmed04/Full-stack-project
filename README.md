# 🍔 Full Stack Food Ordering App

A full-stack web application built using **React** for the frontend and **Laravel** for the backend.
The system allows users to browse menu items, register/login, and place bookings/orders.

---

## 🚀 Tech Stack

* **Frontend:** React.js
* **Backend:** Laravel (PHP)
* **Database:** MySQL
* **API:** RESTful API

---

## ✨ Features

* 🔐 User Authentication (Register / Login)
* 🍽️ Browse Menu Items
* 📅 Booking System
* 👨‍💼 Admin Panel for managing menu items
* 🔔 Notifications system

---

## 📁 Project Structure

```
full-stack-food-app/
│
├── frontend/   # React application
└── backend/    # Laravel API
```

---

## ⚙️ Installation & Setup

### 🔹 Backend (Laravel)

```bash
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

---

### 🔹 Frontend (React)

```bash
cd frontend
npm install
npm start
```
